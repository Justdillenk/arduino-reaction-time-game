# arduino-reaction-time-game
Arduino-based reaction time game using a 4-digit 7-segment display and Arduino Mega 2560

## Overview
This project is a reaction time tester when connected to power there is a countdown using 3 red LED's and after a random interval of time a Green LED lights up and a timer starts. There is a button which will end the timer and show the user their reaction time based on the inteval between the green light shone and the button pressed.

## Hardware Used
-4 Digit Display
-Resistors
-A button
-4 LED's (3 Red and 1 Green)
-an Arduino Mega 2560
-Breadboard
-Jumper Wires

## How to run
Wire how shown in the schematics folder upload to arduino mega 2560 and power on and play
